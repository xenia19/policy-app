Политика конфиденциальности.

Настоящий документ «Политика конфиденциальности» (далее по тексту – «Политика») представляет собой правила использования данных интернет-пользователей (далее «вы» и/или «Пользователь»), собираемых с использованием приложения "Психосоматика | Аффирмации для здоровья".

1. Обрабатываемые данные

1.1. Мы не осуществляем сбор ваших персональных данных с использованием приложения "Психосоматика | Аффирмации для здоровья".

1.2. Все данные пользователя хранятся только в телефоне пользователя.

1.3. Данные пользователя не сохраняются в базах данных и облаках.

1.4. Вы осознаете и принимаете возможность утери информации, которые Вы загружаете в приложение. 

1.5. Администрация не несет ответственность за порядок использования данных Пользователя третьими лицами.

2. Информация в данном приложении носит консультативный характер и не заменяет диагностики и лечения профессиональными специалистами.
